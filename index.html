Тенденции в мире JS

Что такое JavaScript?

JavaScript — это реализация спецификации ECMAScript

Спецификация — это удобно

ECMAScript

  • Первая редакция в 1997 году
  • Разработана на основе оригинальной версии JavaScript и JScript

Проблема прошлых редакций

  • Значительный объем изменений
  • Большой промежуток между редакцями
  • Сложность принятия и реализации

Достижение гармонии

  • Новый стандарт каждый год!
  • Новая система именования редакций (ES2015, ES2016, ...)
  • Stage процесс принятия изменений
    • Stage-0 aka «Чучело»
    • Stage-1 aka «Предложение»
    • Stage-2 aka «Черновик»
    • Stage-3 aka «Кандидат»
    • Stage-4 aka «Завершено»

Babel

  • Позволяет использовать новые возможности уже сейчас
  • Трансформирует код в валидный ECMAScript 5
  • Песочница: https://babeljs.io/repl/

JavaScript повсюду

  • Серверная разработка (Node.js)
  • Разработка Desktop-приложений (Electron)
  • Программирование микроконтроллеров (tessel и espruino)
  • Мобильная разработка (Cordova и React Native)

Мобильная разработка

Apache Cordova

Быстро, легко, но непроизводительно

React Native

Производительно как «нативное» приложение, но сложно и не везде подходит

Итоги. Тенденции

  • Наращавание темпа развития
  • Ежегодное расширение возможностей
  • Распространение на другие платформы
  • Расширение поддержки другими языками
